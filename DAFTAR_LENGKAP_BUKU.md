## üîç CARA MENCARI COVER IMAGE

### Opsi 1: Google Images
```
1. Buka: https://images.google.com
2. Search dengan format:
   - "[Judul Buku] [Penulis]" (contoh: "The Great Gatsby F. Scott Fitzgerald")
   - "ISBN: [ISBN-NUMBER]" (contoh: "ISBN: 978-0-7432-7356-5")
3. Pilih gambar yang paling sesuai
4. Download gambar
```

### Opsi 2: Website Penerbit
```
Cek di website resmi penerbit:
- Penguin Books (penguin.co.uk)
- Simon & Schuster (simonandschuster.com)
- HarperCollins (harpercollins.com)
- Bantam (bantamdell.com)
```

### Opsi 3: Book Database
```
- Goodreads: https://www.goodreads.com (search ISBN atau judul)
- Open Library: https://openlibrary.org
- Amazon: https://www.amazon.com (biasanya ada cover preview)
```

### Opsi 4: Generate dari AI
```
Jika tidak menemukan, bisa:
- Gunakan Canva (canva.com) untuk design cover sendiri
- Atau gunakan AI tools seperti DALL-E, Midjourney
- Atau pakai placeholder service
```
