### 1. Download gambar
```
- Buka: https://images.google.com
- Search: "[JUDUL] [PENULIS] book cover"
- Download gambar
- Time: 1 menit

```

```
- Format: nama-buku-slug.jpg

- Time: 30 detik
```

### 3. Upload ke admin
```
- Buka: http://127.0.0.1:8000/admin/books
- Cari buku
- Klik Edit
- Upload gambar
- Klik Update
- Time: 3 menit
```

## üÜò ERROR SOLUTIONS

### "Cover tidak muncul"
```
Ctrl + Shift + Delete (clear cache)
F5 (refresh)
```

### "Folder tidak ada"
```
Buat folder manual: C:\xampp\htdocs\perpus\storage\app\public\covers\
```

### "Upload error"
```
- Cek format (JPG, PNG saja)
- Cek size < 2MB
- Compress di: https://tinypng.com
```

### "Permission denied"
```
Right-click folder storage
‚Üí Properties ‚Üí Security ‚Üí Full Control untuk Users
```
